HTML Element의 display 속성 2가지 : Block / Inline

block

div element는 block element. 항상 새로운 라인(줄)부터 시작함. 전체 가로 폭(width)을 100% 차지함.

Paragraph element도 block element.

inline

a element는 inline element. 새로운 줄(new line)부터 시작하지 않고, 자신이 필요한 영역만큼만 차지함.

p element 내부에 중첩된(nested) span element 는 자신의 영역만 차지함.
